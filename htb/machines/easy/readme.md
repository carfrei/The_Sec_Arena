# Easy Machines

Beginner-level penetration testing challenges. Focus on fundamental concepts and basic exploitation techniques.

## Difficulty Overview

- **Scope:** Single service or basic vulnerability
- **Skills:** Port scanning, service enumeration, basic exploitation
- **Time:** 30 mins - 2 hours
- **Prerequisites:** Linux basics, networking fundamentals

## Target Platforms

- HackTheBox
- TryHackMe (easy rooms)

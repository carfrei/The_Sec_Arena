# Very Easy Machines

Beginner machines focused on foundational concepts. Introduction to basic networking, services, and exploitation techniques.

## Difficulty Overview

- **Time:** 15 - 30 mins
- **Skills:** Basic networking (ping, nmap), telnet, SSH, service enumeration
- **Focus:** Fundamental penetration testing concepts, weak authentication, basic access

## Tips

- Start with ping to verify connectivity
- Use nmap to identify open ports
- Look for weak/default credentials
- Check for unencrypted services (telnet, ftp)
- Don't underestimate simple solutions

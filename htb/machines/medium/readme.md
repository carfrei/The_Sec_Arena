# Medium Machines

Intermediate-level penetration testing challenges. Require combination of enumeration, exploitation, and privilege escalation.

## Difficulty Overview

- **Scope:** Multiple services or multi-step exploitation
- **Skills:** Advanced enumeration, custom exploitation, privilege escalation, lateral movement
- **Time:** 2 - 6 hours
- **Prerequisites:** Easy machine experience, Linux privilege escalation knowledge

## Target Platforms

- HackTheBox
- TryHackMe (medium rooms)

## Typical Workflow

1. **Reconnaissance:** Full port and service enumeration
2. **Enumeration:** Web application, API, custom protocols
3. **Initial Access:** Exploitation of discovered vulnerability
4. **Privilege Escalation:** Local enumeration and exploit chain
5. **Post-Exploitation:** Evidence gathering, documentation

- Full TCP/UDP scan: `nmap -p- -sV <target>`

- LinEnum for privilege escalation enumeration
- Keep detailed notes on permissions and services
